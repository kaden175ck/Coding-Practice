# VS Code 常用快捷键备忘录

## 🧭 界面控制

- **Ctrl + B**：收起 / 展开文件栏（Explorer）
- **Ctrl + P**：快速打开文件 / 命令
- **Zen Mode（禅模式）**：Ctrl + K，然后按 Z → 进入无干扰模式
- 拖拽文件到编辑区，可以并排对比两个文件的代码

---

## 📝 编辑操作

- **Ctrl + Shift + K**：删除当前行
- **Ctrl + K 0**：折叠所有代码块
- **Ctrl + Tab**：快速在打开的标签页中切换（上下翻页）

---

## 🖥️ 终端与预览

- **Ctrl + J**：显示 / 隐藏终端
- **Ctrl + Shift + V**：Markdown 预览
- **Ctrl + K V**：Markdown 侧边预览

---

## ✍️ 多重编辑 / 批量修改

- **Ctrl + D**：选中下一个相同的匹配项（可连续按，多选）
- **Ctrl + Shift + L**：一次性选中所有匹配的内容，适合批量修改相同变量名
